# Shower System and Waterproofing

## Outcome

Define one complete, compatible shower assembly that reliably controls water from the framing through the finished tile.

## Decisions Required

- Which complete waterproofing system will be used?
- Which compatible drain and shower-base method will be used?
- Who will install and warranty the system?
- How will walls, corners, seams, fasteners, penetrations, curb, and storage features be waterproofed?
- How will the required shower-floor slope be created and verified?
- What flood-test method and duration are required?
- What is the stop-work process if demolition reveals damage, mold, or structural problems?

## Recommended MVP

- Select one proven manufacturer system and use compatible components throughout
- Use a conventional curbed entry unless accessibility or layout goals justify the additional complexity of curbless construction
- Remove the bench and avoid a niche if either feature complicates waterproofing or strains the budget
- Require a documented flood test before tile installation

## Non-Negotiables

- Tile, grout, and caulk are not the primary waterproof barrier
- The subfloor and framing must be sound before rebuilding
- The shower floor must drain completely without pooling
- Every seam, transition, and penetration must follow the selected system's instructions
- Waterproofing must pass inspection and flood testing before being covered

## Dependencies

- Final shower layout and drain plan
- Plumbing fixture selections
- Tile type and format must be compatible with the selected shower base

## MVP Exit Criteria

- [ ] Complete waterproofing system selected
- [ ] Compatible drain and shower-base strategy selected
- [ ] Installer and warranty responsibilities confirmed
- [ ] Required slope and flood-test procedure documented
- [ ] Post-demolition inspection checklist agreed

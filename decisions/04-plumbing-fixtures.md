# Plumbing and Fixtures

## Outcome

Select plumbing fixtures and confirm rough-in requirements before walls and floors are closed.

## Decisions Required

- Will shower plumbing remain in its current location?
- What shower valve, controls, head, and optional hand shower will be used?
- Does the selected drain work with the waterproofing system?
- Which vessel sinks fit the existing countertop cutouts?
- Which faucets fit the sinks, countertop, and existing or planned hole pattern?
- Will shutoffs, supply lines, or drain piping be replaced?
- Are any plumbing upgrades required after demolition?

## Recommended MVP

- Keep existing plumbing locations when practical
- Use a simple, serviceable shower configuration
- Select fixtures before rough-in begins
- Replace inaccessible or questionable plumbing while walls are open

## Dependencies

- Measurements and layout
- Shower waterproofing system
- Finish direction for visible fixture color

## MVP Exit Criteria

- [ ] Shower valve and fixture package selected
- [ ] Compatible shower drain selected
- [ ] Vessel sinks and faucets confirmed against measurements
- [ ] Rough-in locations documented
- [ ] Plumbing inspection and testing plan confirmed
